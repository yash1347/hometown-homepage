## Overview
This is a simple and responsive website created using **HTML** and **CSS** to showcase the beauty, culture, history, and famous attractions of Jaipur, Rajasthan. The website provides visitors with information about Jaipur's heritage, architecture, cuisine, and tourism highlights.

## Features
- Responsive and clean user interface
- Information about Jaipur's history and culture
- Showcase of famous tourist attractions
- Attractive layout and styling using CSS
- Easy navigation between sections
- Mobile-friendly design

## Technologies Used
- HTML5
- CSS3
